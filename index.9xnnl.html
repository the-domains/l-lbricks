Innovative Braille LEGO-Style Bricks Help Blind Children Learn To Read While Playing

The Braille Bricks project, co-founded by the Dorina Nowill Foundation For The Blind, helps blind children learn to read through play. Each block can be used like a toy by blind and sighted children alike, although for the benefit of the blind children, each block features one Braille letter.
